This is a desktop application built with Electron, Vue 2, Bootstrap and BootstrapVue. It includes a secure bridged api endpoint as a starting point for providing integrations without exposing user data, e.g. API_KEYS, to 'world facing' render processes. See Electron's process model and Electron's inter process communication

We are using Node.js , Chromium , and Electron .
